## 📖 About The Project

A sleek, modern, and fully responsive landing page built entirely with pure HTML and CSS. This project heavily utilizes the **Glassmorphism** design trend, combining frosted glass UI components with smooth, continuous animations and engaging micro-interactions. It serves as a perfect template for personal portfolios, product showcases, or modern web interfaces.

## ✨ Key Technical Features

This repository demonstrates several modern frontend design techniques:

* **Glassmorphism UI:** Advanced use of `backdrop-filter: blur(15px)`, semi-transparent `rgba` backgrounds, and subtle white borders to create a realistic frosted glass effect for the header and main content box.
* **Continuous Floating Animation:** The main feature image (Steve Jobs) utilizes custom CSS `@keyframes` (`translateY`) to create a smooth, continuous floating illusion.
* **Fully Responsive Layout:** Implements CSS media queries (`@media (max-width: 1000px)`) to gracefully adapt the flexbox layout, header positioning, and animation distances for mobile and tablet devices.
* **Smooth Micro-interactions:** Social media icons feature a clean `rotate(360deg)` and scaling transition on hover to enhance user engagement.

## 🚀 How to Use

You can easily clone and use this modern landing page for your own projects:

1. Clone the repository:
```bash
git clone https://github.com/Hosein-raz/steve-landing.git
```
2. Open the `index.html` file in your browser to see the page in action.
3. Replace the images in the `images/` folder with your own assets and modify the text in the HTML file to personalize it.

## 📂 Project Structure

```
steve-landing/
├── images/       # Contains background, main character, and social icons
├── index.html    # The semantic HTML structure and content
└── style.css     # Glassmorphism logic, media queries, and keyframe animations
```
